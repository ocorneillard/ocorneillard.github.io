C is a call-by-value language ; when you provide an argument to a function, the value of that argument is copied into a distinct variable for use within the function. Coming from Javascript, or any higher level language, and it's a new paradigm to understand, because it's possible to optimize something that was given for free earlier, but to be honest, it should be done by the compiler and focusing on Big O to understand the speed of algorithm is probably better. Yet I still want to learn C.
